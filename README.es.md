# Code Ghost

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Skill de Code Ghost para OpenClaw.

## Problema que resuelve
Revela patrones ocultos y anti-patrones en bases de código.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque primero en seguridad
- Soporte para rollback

## Uso
/code-ghost

## Ejemplo
```
Analizar código para detectar problemas de análisis.
```

## Instalación
Esta skill está disponible automáticamente en OpenClaw.
